🔗 技术关系概览

运行时层

Bun 提供快速 TypeScript 运行时，内置打包与原生模块支持， 替代传统 Node.js + webpack 组合。

UI 渲染层

React + Ink 将组件化思维引入终端界面， 实现响应式布局和实时流式渲染。

AI 集成层

Anthropic SDK 驱动模型调用，Zod 负责工具参数校验， MCP 协议标准化外部工具集成。

搜索与性能层

ripgrep（Rust 实现）提供超高速代码搜索， Rust crates 承担部分高性能计算任务。
